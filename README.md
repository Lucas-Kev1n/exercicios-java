# Exercícios de Programação Orientada a Objetos em Java

Repositório criado para praticar os fundamentos de POO aplicados com Java puro.

## 🧠 Lógica de Programação Básica Praticada
Antes de avançar para a Orientação a Objetos, consolidei os conceitos fundamentais de lógica utilizando Java:

- **Estruturas Condicionais (`if`, `else if`, `else`):** Controle de fluxo para tomadas de decisão no sistema.
- **Estruturas de Repetição (`while`, `for`):** Manipulação de loops para iterações e automação de processos.
- **Entrada e Saída de Dados:** Manipulação do console utilizando `Scanner` para leitura e `System.out.printf` para formatação de dados (como formatação de moedas com casas decimais).
- **Operadores Aritméticos e de Atribuição:** Aplicação de lógica matemática (somas, subtrações, acumuladores como `+=` e `-=`).

## 🚀 Conceitos Praticados no Desafio da Conta Bancária:
- **Encapsulamento:** Atributos protegidos com `private` e controle de acesso via Getters e Setters estratégicos.
- **Sobrecarga de Construtores:** Permite a criação de contas com ou sem depósito inicial informado.
