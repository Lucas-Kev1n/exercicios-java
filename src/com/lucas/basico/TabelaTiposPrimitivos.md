| Tipo | Descrição                 | Tamanho | Valores (Intervalo) | Padrão |
| :--- |:--------------------------|:--------| :--- | :--- |
| **byte** | Inteiro                   | 8 bits  | -128 a 127 | `0` |
| **short** | Inteiro                   | 16 bits | -32.768 a 32.767 | `0` |
| **int** | Inteiro                   | 32 bits | -2.1 bi a 2.1 bi | `0` |
| **long** | Inteiro                   | 64 bits | Muito grande | `0L` |
| **float** | Numérico ponto flutuante  | 32 bits | 6-7 casas decimais | `0.0f` |
| **double** | Numérico ponto flutuante  | 64 bits | 15 casas decimais | `0.0d` |
| **char** | Caractere único (Unicode) | 16 bits | Unicode (ex: 'A') | `'\u0000'` |
| **boolean** | Lógico - Valor verdade    | 1 bit   | `true` ou `false` | `false` |
